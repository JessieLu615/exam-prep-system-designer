# Planning Framework

Planning converts goals into weekly and daily action. The plan should guarantee coverage before the exam while leaving room for adjustment.

## Inputs

Use:
- Remaining days.
- Target scores or target result.
- Current subject scores/progress.
- Guaranteed study time.
- Materials and required coverage.
- Weak modules.
- Fixed review requirements.

## Build The Total Plan

1. Split the remaining days into phases:
   - Foundation or first framework pass.
   - Reinforcement and question practice.
   - Mock exam and correction.
   - Final recall and stabilization.
2. Assign each subject a role:
   - Growth subject: needs score increase.
   - Maintenance subject: needs regular preservation.
   - Risk subject: repeatedly unstable, gets fixed review.
3. Define measurable units:
   - Chapters/framework nodes.
   - Practice sets.
   - Wrong-question reviews.
   - Subjective-answer records.
   - Memorization/review blocks.
4. Reserve buffer:
   - Keep at least one weekly buffer block.
   - Do not plan every unstable weekday at maximum hours.

## Weekly Plan

A weekly plan should include:
- Core subject work.
- Memorization and review.
- Practice.
- Wrong-question review.
- One weekly review and adjustment.
- Backup tasks for low-time days.

Use a three-level task structure:
- Must finish: minimum viable progress.
- Should finish: normal target.
- Stretch: only if time and energy allow.

## Daily Plan

Daily tasks should be short enough to execute:
- 2 to 5 tasks for a normal day.
- Each task should have a measurable output.
- Include one feedback item at night: actual study time, completed tasks, biggest problem.

Example:
- 312: finish one chapter-level mind map.
- 312: complete 20 matching multiple-choice questions.
- Memorization: create 10 to 15 knowledge cards from today's framework.
- Politics: new module framework for 1 hour.
- Evening feedback: actual time, completed items, blocker.

## Progress Tracking

Track progress with counts, not vague feelings:
- Completed tasks / total tasks.
- Chapters completed / chapters planned.
- Wrong questions pending review.
- Subjective answers recorded.
- Consecutive review days.

The dashboard should show enough progress to reduce anxiety, but not so much that it becomes a second exam.
